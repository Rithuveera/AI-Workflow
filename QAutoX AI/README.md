# QAutoX

**"Automate. Analyze. Accelerate."**

QAutoX is an AI-powered Quality Assurance platform designed to streamline the entire testing lifecycle. From generating test cases from requirements to automating execution scripts and analyzing defects, QAutoX brings intelligence to every step of your QA process.

- [Main Documentation](docs/AI_TESTING_PLATFORM_DOCUMENTATION.md)
- [API Documentation](docs/API_DOCUMENTATION.md)

## How to Run

1.  **Install Dependencies** (if not already installed):
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the Application**:
    ```bash
    python run.py
    ```

3.  **Access the App**:
    Open [http://localhost:5000](http://localhost:5000) in your browser.

## Project Structure
- `app/`: Main application package (Routes, Models, Services)
- `docs/`: Project documentation
- `scripts/`: Utility scripts
- `tests/`: Test scripts
- `run.py`: Entry point
